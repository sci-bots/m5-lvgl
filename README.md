# Install

The latest [`m5-lvgl` release][1] is available as a
[Conda][2] package from the [`sci-bots`][2] channel.

To install `m5-lvgl` in an **activated Conda environment**, run:

    conda install -c sci-bots -c conda-forge m5-lvgl
